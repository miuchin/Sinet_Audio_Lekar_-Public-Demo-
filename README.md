# SINET Audio Lekar — Public Demo

**Language / Jezik:** [Srpski (README_SR.md)](README_SR.md) · [English (README_EN.md)](README_EN.md)

SINET Audio Lekar is a **PWA/offline-first demo** for structured audio routines, symptom catalog navigation, personal lists/protocols, and auxiliary tools (Anamneza, DS-Generator, Integrative Guide, AI Questionnaire, Admin Tools).

## Quick links
- **Main app:** `index.html`
- **No Service Worker test mode:** `index-nosw.html`
- **User manual (SR, HTML):** `SINET_User_Manual_v1.1_SR.html`
- **Tutor / Vodiči hub (SR, HTML):** `docs/protokoli/00_TUTOR_VODICI_INDEX_v1.0_SR.html`

## What is included
- Main SINET Audio Lekar app (PWA)
- Symptom catalog + presets / quick help
- Personal lists (Queue), favorites, custom protocols
- Anamneza + MKB related tools
- Admin toolchain (converter, deduplicator, linker, dx index generator, inspector)
- Tutor/Protocol docs (SR, HTML + MD)
