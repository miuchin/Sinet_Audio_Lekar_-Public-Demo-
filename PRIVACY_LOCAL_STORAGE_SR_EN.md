# Privacy & Local Data Note (SR / EN)

## SR
Ova demo aplikacija radi pretežno lokalno (browser/PWA) i koristi lokalno čuvanje podataka (npr. localStorage / IndexedDB) za:
- Favoriti
- Queue / Lista
- Moji protokoli
- Podešavanja
- Lokalni override fajlovi (gde je primenljivo)

Backup/Restore funkcije izvoze/uvoze fajl **na uređaju korisnika**.  
Proverite sadržaj backup fajla pre deljenja sa drugima.

## EN
This demo app is primarily local/offline (browser/PWA) and uses local storage (e.g. localStorage / IndexedDB) for:
- Favorites
- Queue / List
- My Protocols
- Settings
- Local override files (where applicable)

Backup/Restore exports/imports files **on the user device**.  
Review backup content before sharing it with others.
