# GitHub Release Checklist (SR / EN)

## SR — Pre objave
- [ ] Proveri `index.html` i `index-nosw.html`
- [ ] Testiraj osnovni playback (desktop + 1 mobilni uređaj)
- [ ] Testiraj Menu / Katalog / Podešavanja / Play kontrole
- [ ] Proveri Tutor / Vodiči linkove (HTML)
- [ ] Proveri `service-worker.js` bez syntax error-a
- [ ] Ažuriraj `CHANGELOG.md`
- [ ] Ažuriraj verziju u UI (ako je potrebno)
- [ ] Commit + tag (`vX.Y.Z`)
- [ ] GitHub Release notes
- [ ] Netlify deploy check

## EN — Before release
- [ ] Verify `index.html` and `index-nosw.html`
- [ ] Test basic playback (desktop + one mobile device)
- [ ] Test Menu / Catalog / Settings / player controls
- [ ] Verify Tutor/Guides HTML links
- [ ] Confirm `service-worker.js` has no syntax errors
- [ ] Update `CHANGELOG.md`
- [ ] Update UI version (if needed)
- [ ] Commit + tag (`vX.Y.Z`)
- [ ] GitHub Release notes
- [ ] Netlify deploy check
