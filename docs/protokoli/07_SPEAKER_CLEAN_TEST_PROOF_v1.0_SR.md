# SINET — PROTOKOL: Speaker Clean + Test + Proof (Mobile) (v1.0, SR)

**Uloga:** da proveriš i očistiš zvuk zvučnika (posebno na telefonu), uz test i “proof” korake.

> ⚠️ Bezbednost: ne pojačavaj zvuk ekstremno dugo. Ako čuješ krčanje, odmah smanji.

---

## 1) Step-by-step (najbolja praksa)

1. Otvori **Speaker Clean + Test + Proof**
2. Spusti glasnoću na ~50% (start)
3. Klikni **CLEAN** (kratko)
4. Klikni **TEST** (stereo / rattle)
5. Ako postoji **PROOF** (mikrofon): uradi proof da potvrdiš rezultat
6. Ako se čuje bolje: ponovi CLEAN 1–3 puta (kratko, sa pauzom)

---

## 2) Polje-po-polje / dugme-po-dugme

- **CLEAN (desni/levi/oba)**: pušta signal za izbacivanje prljavštine/tečnosti iz rešetke
- **TEST**: proverava balans, rattle, frekvencije koje otkrivaju problem
- **PROOF**: koristi mikrofon (gde je dostupno) da izmeri i potvrdi promenu

---

## 3) Mehaničko čišćenje (preporučeno uz audio)

1. Ugasiti zvuk
2. Mekana četkica + blago tapkanje
3. Blu‑Tack (lepljiva masa) — nežno pritisni i podigni (ne guraj unutra)
4. Tek onda uradi CLEAN/TEST

---

## 4) Troubleshooting

- **Ne radi PROOF** → browser ne daje mic dozvolu / file:// okruženje; koristi HTTP server ili PWA
- **Zvuk nema efekta** → moguće fizičko oštećenje ili voda dublje; potraži servis
- **Bluetooth slušalice** → testiraj i preko ugrađenog zvučnika (da vidiš razliku)
